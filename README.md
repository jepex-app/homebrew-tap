# homebrew-tap

Homebrew tap for [jepex](https://jepex.app) — a JPEG photo editor and library
for straight-out-of-camera shooters on Fujifilm, Ricoh, and Nikon.

## Install

```sh
brew install --cask jepex-app/tap/jepex
```

Once installed, jepex keeps itself up to date via its built-in updater.

## Uninstall

```sh
brew uninstall --cask jepex
```
